### 실행 방법
```bash
# Ollama 실행 및 Nomic 임베딩 모델 다운로드
docker run -d -v ollama:/root/.ollama -p 11434:11434 --name ollama ollama/ollama
docker exec -it ollama ollama pull nomic-embed-text-v2-moe

# 가상환경 활성화 후 의존성 설치
pip install -r requirements.txt

# (.env 파일에 GROQ_API_KEY 세팅 확인 후)
# 메인 스크립트 실행
python app.py
```

Q: 피치클락 위반 시 타자와 투수에게 각각 어떤 페널티가 부여되나요?
피치클락 위반 시는 규정 ID 5 (타자의 타석 이행 의무)과 규정 ID 4 (피치클락 제한 시간 규정)가 관련이 있습니다.

규정 ID 5 (타자의 타석 이행 의무)에 따르면, 타자는 피치클락 잔여 시간이 8초가 되기 전까지 타석에 들어와 투구 준비를 마쳐야 하며 위반 시에는 스트라이크 1개가 선언됩니다.      

규정 ID 4 (피치클락 제한 시간 규정)에 따르면, 투수는 주자가 없을 때 18초, 주자가 있을  때 23초 이내에 투구 동작을 시작해야 하고 위반 시에는 볼 1개가 선언됩니다.

따라서 피치클락 위반 시에는 타자는 스트라이크 1개가 선언되고 투수는 볼 1개가 선언됩니다.
--------------------------------------------------
Q: A선수가 3일 전 경기에 나갔고 오늘 부상으로 말소됐다면, 소급 적용을 포함해 언제 복귀 가능한가?
ID 9 규정에 따르면, 엔트리 말소일로부터 최대 3일 전까지 소급하여 부상자 명단 기간을 산 정할 수 있다. 단, 해당 기간 중 경기에 출전한 사실이 없어야 한다.

A선수가 3일 전 경기에 나갔기 때문에, 소급 적용을 포함해 부상자 명단 기간을 산정할 수 없다. 이 경우, 부상자 명단 기간은 엔트리 말소일부터 시작되며, 복귀는 엔트리 말소일로부터 10일이 경과해야 할 것이다.

따라서 A선수의 복귀 가능 날짜는 엔트리 말소일로부터 10일 후이다.
--------------------------------------------------
Q: 경기 중 ABS가 고장 났고, 5회초에 비가 내려 경기가 중단된 상황입니다. 현재 경기의 최 종 상태(종료/일시중단)는 어떻게 판정되나요?
일시중단 상황입니다. 

규정 ID 2에 따르면, 기술적 결함으로 ABS 판정 수신이 불가능할 경우, 해당 투구에 한해 주 심의 자체 판정으로 대체하며 양 팀 감독에게 즉시 고지한다.

규정 ID 2는 ABS 추적 실패 시 주심의 자체 판정으로 대체하도록 규정하고 있기 때문에, ABS 가 고장 났을 경우에도 주심의 자체 판정으로 대체해야 합니다.

규정 ID 24에 따르면, 강우 콜드게임은 5회말 홈팀 공격이 끝났거나 5회초 원정팀 공격 종료 시점에 홈팀이 앞서있을 때 경기가 중단된 경우 성립된다. 현재 상황은 5회초에 강우콜드가  발생한 상태이므로, 경기는 일시중단 상태입니다.

따라서, 현재 경기의 최종 상태는 일시중단 상태이며, 강우 콜드게임은 성립되지 않습니다.  
--------------------------------------------------
